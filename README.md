## Proyectos Principales

Plataforma de reportes comunitarios enfocada en la gestión de información y procesamiento de datos para la administración de incidencias.  
 Tecnologías: HTML, CSS, JavaScript, Java

---

### Simmer
Aplicación para compartir recetas con funcionalidades de interacción, enfocada en la gestión de usuarios y almacenamiento de datos.  
 Tecnologías: Kotlin, Firebase

---

### Voucher
Aplicación web para la gestión de ingresos y gastos personales, centrada en el manejo de datos y lógica de negocio para la organización financiera.  
 Tecnologías: Next.js, PostgreSQL
